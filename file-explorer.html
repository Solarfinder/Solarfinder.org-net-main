Load from Relative URL

Enter a relative path to a folder. The system will look for a manifest.json file in that folder describing the structure or load files directly.

Example: manifest.json file structure

{ "name": "My Folder", "files": [ {"name": "document.pdf", "path": "/assets/documents/document.pdf"}, {"name": "image.jpg", "path": "/assets/documents/image.jpg"} ], "folders": [ { "name": "subfolder", "files": [ {"name": "file.txt", "path": "/assets/documents/subfolder/file.txt"} ] } ] }
